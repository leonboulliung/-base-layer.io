# BaseLayer Landing Page - PRD

## Original Problem Statement
Create a high-conversion landing page for a B2B SaaS product targeting coliving / shared housing operators. The goal is to generate demo calls by clearly communicating two core outcomes:
1. Reduce friction in the resident experience
2. Control and reduce operational cost per unit

## Product Details
- **Product Name:** BaseLayer (base-layer.io)
- **Demo Booking:** Calendly integration (https://calendly.com/leon-boulliung-grabitautomaten/leon-call)

## Target Audience
- Coliving operators
- Property managers
- City managers
- Companies managing shared apartments or buildings

## User Personas
1. **Property Manager** - Manages 5-50 units, overwhelmed with WhatsApp coordination, seeking operational efficiency
2. **Coliving Operator** - Runs multiple coliving spaces, wants to reduce cost per unit and improve resident satisfaction
3. **City Manager** - Oversees large portfolio, needs visibility across properties and standardized operations

## Core Requirements (Static)
- Clear value proposition: "Reduce friction. Control cost per unit."
- Professional B2B design (real estate + tech aesthetic)
- Calendly demo booking integration
- Mobile-responsive design
- FAQ section to reduce conversion friction
- Smooth scroll navigation
